In this game, the user controls a `shadow` and is tasked with following the `fly` as closely as possible.

There are three modes in this game. The game will suddenly switch from one mode to the next and the user needs to learn and adjust to the change
as quickly as possible.

The first mode is "normal". Normal mouse/trackpad/etc. movements will translate to the expected movements of the `shadow`. In the other modes, 
this is not necessarily so.

Do note, that your data (metadata like screen dimension and any information submitted including cursor position) is collected and saved to a Firestore database.
